WinMerge User Documentation

This folder contains WinMerge's user- documentation.

User Documentation

These documents are meant for casual users. They help users with WinMerge use, tell about changes in WinMerge versions. Some files are included in the installer, but not all. More files are included in binary distribution, but not all files there either.

User's Manual:
User's manual is not directly readable from our source tree. It must be first converted to HTML or HTML help. See Developer-documentation for instructions how to do it.

Other files in user documentation:

  • ReadMe is shown in end of the installer and is installed into Windows Start-menu also. It lists most important links for help, installing archive support etc.
  • ChangeLog contains general list of changes between WinMerge versions.
  • ReleaseNotes has release-specific information, like known issues.
  • Contributors list lists people in development group and other people who have helped in development.
  • Files list lists files WinMerge installs.
  • GPL.rtf contains GPL license in RTF format.
